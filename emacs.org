#+Title: Emacs

* MINI HOWTOS

** Insert any UTF char

Use M-x  =insert-char=

** Set C Coding Style In Emacs

In a programming mode, issue the command

   c-set-style

Possible completions are:

  bsd
  cc-mode
  ellemtel
  gnu           <--- default style
  java
  k&r
  linux         <--- C coding standard for Linux development.
  python        <--- C coding standard for Python extension modules

  stroustrup    <--- USE THIS?

  user
  whitesmitht
